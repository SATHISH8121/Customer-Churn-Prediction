# Customer Churn Prediction

## Overview
This project predicts whether a telecom customer will leave the company using Machine Learning.

## Dataset
- Telco Customer Churn Dataset
- 7043 records
- 21 features

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Models Tested
- Logistic Regression (81.48%)
- Random Forest (79.70%)

## Final Model
Logistic Regression

## Results
Achieved 81.48% accuracy.
