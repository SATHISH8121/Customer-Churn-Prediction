# 📌 Project Overview

This project predicts whether a telecom customer is likely to leave (churn) or stay with the company using Machine Learning algorithms.

---

# 🎯 Objective

Build a predictive model that helps telecom companies identify customers who are likely to churn so that proactive retention strategies can be implemented.

---

# 📂 Dataset

- **Dataset:** Telco Customer Churn Dataset
- **Records:** 7,043
- **Features:** 21

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/Customer-Churn-Prediction.git
```

## Create Virtual Environment

```bash
python -m venv venv
```

## Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```
