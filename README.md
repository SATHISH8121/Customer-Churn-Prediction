# 📊 Customer Churn Prediction

# 📌 Project Overview

This project predicts whether a telecom customer is likely to leave (churn) or stay with the company using Machine Learning algorithms.

---

# 🎯 Objective

Build a predictive model that helps telecom companies identify customers who are likely to churn so that proactive retention strategies can be implemented.

---

# 📂 Dataset

- **Dataset:** Telco Customer Churn Dataset
- **Records:** 7,043
- **Features:** 21
